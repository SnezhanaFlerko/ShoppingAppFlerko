# ShoppingAppFlerko

## Описание проекта:
### Приложение для Android, реализующее функцию списка покупок с возможностью:
• Добавления товаров (название, количество, цена)
• Отметки товара как купленного
• Установки напоминаний о покупке на определённое время
• Переключения между светлой и тёмной темой
